# Super Mario Pixel Adventure 🍄🎮

A visually stunning, retro 8-bit platformer and interactive tribute built with **React**, **Vite**, and **Tailwind CSS**. Experience classic side-scrolling, custom difficulty settings, cheat codes, authentic sound synth effects, and an epic boss showdown with Bowser!

---

## 🌟 Key Features

* **Retro 8-Bit Gameplay**: Classic movement physics, jumping, brick bashing, coin collection, power-ups, and pipes.
* **Bowser Boss Showdown**: Face off against Bowser on a traditional bridge. Dodge alternating fire breaths, jump over heavy stomps, and drop him into the lava below using the Golden Axe trigger.
* **Visual Effects**: Screen shake on Bowser stomps and bridge collapse, custom dust clouds, and hand-crafted HTML/CSS pixel elements.
* **Audio Synthesizer Engine**: Dynamic sound effects generated programmatically for jumps, brick breaking, power-up collections, fireballs, and defeats.
* **Custom Customization Overlays**: 
  * Simple Cheat system (passcode unlockable menu).
  * Smooth difficulty switches (Normal, Hard, Impossible, etc.).
* **Fully Responsive**: Playable on mobile devices with retro visual button overrides or on desktops using standard keyboard navigation.

## 🎮 Game Controls

### ⌨️ Desktop (Keyboard)
* **Arrow Left / D / A**: Move left
* **Arrow Right / D**: Move right
* **Spacebar / Z**: Jump / High Jump
* **X**: Shoot Fireball (when powered up by Fire Flower)
* **Arrow Down / S**: Duck

### 📱 Mobile (Touch Controls)
* **On-Screen D-Pad (Left/RightArrows)**: Movement
* **Button A**: Jump (Hold longer for higher jumps)
* **Button B**: Shoot Fireball
